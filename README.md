# Pokemon Math Adventure

A web-based math puzzle game for kids where solving math problems earns Pokeballs to capture and evolve Pokemon.

## How to Play

Open `index.html` in your browser - no server or installation required.

## Features

- 8 math topics: Addition, Subtraction, Skip Counting, Shapes, Visual Grouping, Place Value, Fractions, Word Problems
- 8 collectible Pokemon with evolution chains
- Parent-controlled difficulty settings (Easy/Medium/Hard)
- Offline play with LocalStorage progress saving
